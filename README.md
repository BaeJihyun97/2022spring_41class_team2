# 프로그램 세팅 방법

## 버전
- Node : 17.5
- Python : 3.7

### 파이썬 가상환경을 만들고 진행하기를 권장합니다.

## 프론트엔드 세팅 방법
```shell
cd client
npm install .
```

## 벡엔드 세팅 방법
```shell
cd server
pip install .
```

## 실행 세팅 방법
```shell
npm install concurrently
```

## 프로그램 실행 방법
```shell
npm run dev
```
